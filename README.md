# Smart Robotic Rover for Precision Weed Surveillance and Removal

## Overview
An ESP32-CAM based agricultural rover for real-time weed monitoring and remote weed removal.

## Features
- Live Video Streaming
- Wireless Rover Control
- Weed Cutting Mechanism
- Camera Pan & Tilt
- Speed Control
- Lighting Control

## Hardware Used
- ESP32-CAM
- L298N Motor Driver
- LM2596 Buck Converter
- Relay Module
- DC Motors
- 12V Battery

## Software
- Arduino IDE
- Embedded C/C++

## Working
The rover creates a Wi-Fi access point and hosts a web interface. The user connects through a smartphone or laptop, views live video, controls movement, and activates the cutter remotely. 

## Future Enhancements
- AI Based Weed Detection
- Autonomous Navigation
